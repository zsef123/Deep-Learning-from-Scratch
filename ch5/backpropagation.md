Chatper 5. 오차 역전파법
=============================
가중치 매개변수의 기울기르 효율적으로 계산한다.

## 계산 그래프Computation graph

복수의 node, edge 로 표현된다.
100원인 사과 2개를 샀을때 지불 금액을 구하라 이때 소비세 10%가 부과된다. 
= (사과) -> (*2) -> (*1.1) -> (금액)
= (사과) -> (*)  -> (*) -> (금액)
             ㄴ 개수  ㄴ 소비세

계산그래프를 구성하여 순방향Foward로 진행한다.
+ 국소적 계산을 전파함으로써 최종 결과를 얻는다.
